4. **协议帧格式全面重写（回复数据帧）**：
   - 旧格式: [EB90 2B][版本1B][长度4B][命令2B][payload N][CRC16-CCITT 2B] = 变长帧
   - 新格式（来自规格图）:
     | 字段 | 长度 | 说明 |
     |------|------|------|
     | 帧头 | 2B (16Bits) | 固定 0xEB90 |
     | 帧类型 | 1B (8Bits) | 固定 0x04 |
     | 指令类型 | 1B (8Bits) | 如 0xC1=遥测数据, 0xA1=心跳等 |
     | 数据域 | **1024B** (8192Bits) | 遥测有效数据，固定1024字节 |
     | 校验和 | 1B (8Bits) | **数据域累加校验和**（非CRC16，仅对数据域求和取低8位）|
   - 总帧长：**固定 1029 字节**
   - 关键变化：
     - 命令码从 quint16 → **quint8**（指令类型）
     - CRC16-CCITT → **8位累加校验和**
     - 数据域从变长 → **固定 1024 字节**
     - payload 不足 1024 时自动补零
   - 更新文件: dataprotocol.h/cpp 全面重写, mainwindow.h/cpp 中 sendFrame/broadcastFrame/commandName 签名从 quint16→quint8, hasMinimumFrameSize→hasCompleteFrame
