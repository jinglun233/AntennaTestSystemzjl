## 遥测子帧解析链路接入 + ElectronicDeviceWindow 控件映射修正
### 新增方法 (mainwindow.cpp/h)
- **FindData**: 从 RingBuffer 搜索帧头 0xEB90，提取完整1029字节帧
- **Validate**: 累加和校验（对应C# Validate）
- **ResolveData**: FindData → 非零判断 → Validate → 子数据段拆分（从索引11开始按 len 解析）
- **JudgeData**: 遍历子段判断类型：0x3A+0xFF→updataYC(波控), 0x51AA→updataDir(电子设备)
- **updataYC**: 转发速变遥测到 AntennaDeviceWindow
- **updataDir**: 转发直接遥测到 ElectronicDeviceWindow::decodeTelemetry

### 数据流

### decodeTelemetry 控件映射重写 (electronicdevicewindow.cpp)
- NOM解码器: Calc字段idx → cgYCline_{idx+1}（后缀=索引+1）

## FindData 重写 — 适配 C# 遥测子帧协议（0x68 帧头）
- **帧协议变更**: 从 EB90 标准帧改为 0x68 帧头的遥测子帧协议
  - 帧头格式: `[0x68] [len_H] [len_L] [0x68]`，索引0和索引3都必须是0x68
  - 数据长度: `leng = idx[1]*256 + idx[2]`，有效范围 7~1017
  - 总帧长: `leng + 7`
- **实现方式**: 用 RingBuffer 的 `peek()` 返回 QByteArray 后按 `preview.at(n)` 索引访问，`read(leng+7)` 批量提取完整帧
- **异常处理**: 添加 try/catch 包裹 + qWarning 日志输出
