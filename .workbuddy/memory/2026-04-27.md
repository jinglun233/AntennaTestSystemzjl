### 自动化测试流程骨架
**流程设计**（状态机驱动，QTimer 非阻塞延迟）：
```
for each 通道 in configTableWidget第二列:
  ├─ ReadConfig → 提取通道名列表
  ├─ DownloadWaveCode → 读 波位目录/通道名.txt → 分页下发(50ms间隔)
  ├─ StartPrf → 发送 \xEB\x90\x01\xF1\x05 (50ms后)
  └─ VnaMeasure → SCPI指令序列:
      ├─ :CALC:PAR:SEL + :CALC:PAR:MOD:EXT 'S12' + :DISP:WIND:TRAC:FEED
      ├─ VnaSaveFile → :MMEM:MDIR/:CDIR + :CALC:DATA:SNP:PORTs:Save
      └─ VnaSaveScreen → :HCOPy:FILE
  ─ NextChannel 或 StopPrf → Complete
```

**新增信号/槽连接**：
| AutoTestWindow 信号 | 接收方 | 方法 |
|---------------------|--------|------|
| waveCodeDownloadRequested(path) | AntennaDeviceWindow | downloadSingleWaveCode() |
| startPrfRequested | AntennaDeviceWindow | startPrf() |
| stopPrfRequested | AntennaDeviceWindow | stopPrf() |
| vnaScpiCommandRequested(scpi) | NetworkManager | sendToVna() |

**AntennaDeviceWindow 新增公共方法**：
- `downloadSingleWaveCode(filePath)`: 单文件波控码分页下发（自动测试用）
- `startPrf()` / `stopPrf()`: PRF 启停控制
