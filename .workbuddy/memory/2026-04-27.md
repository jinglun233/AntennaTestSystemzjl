### 异步波控码下发（c8e6903）
**问题**：`downloadSingleWaveCode()` 内 `QThread::msleep(50)` 同步阻塞，数据量大时 UI 卡死

**改造为 QTimer 事件驱动**：
```
downloadSingleWaveCode(filePath)
  ├─ 文件读取 + 解析二进制波控码
  ├─ sendBokongCode(totalPages, 0xB0)   ← 发送总页数指令
  └─ m_wavePageTimer->start(50)         ← 启动定时器
       │
       ▼ onWavePageTimer() [每50ms触发一次]
       ├─ bokongMaSend(dataPage, page)   ← 发送一页(1024字节)
       ├─ m_wavePageTimer->start(50)     ← 还有下一页？继续
       └─ emit waveCodeDownloadCompleted(ok, totalPages)  ← 全部完毕
            │
            ▼ AutoTestWindow::onWaveCodeDownloadCompleted()
            └─ scheduleNextStep(WaitAfterWaveCode, 50)    ← 推进状态机
```

**新增信号/槽**：
| 信号 | 来源 | 槽 | 说明 |
|------|------|-----|------|
| `waveCodeDownloadCompleted(bool,int)` | AntennaDeviceWindow | AutoTestWindow::onWaveCodeDownloadCompleted | 波控码异步下发完成通知 |
| `waveCodeDownloadRequested(path)` | AutoTestWindow | AntennaDeviceWindow::downloadSingleWaveCode | 请求下发单个文件 |

**新增成员变量（AntennaDeviceWindow）**：
- `QTimer *m_wavePageTimer` — 分页定时器
- `QByteArray m_pendingWaveData` — 待发送数据
- `int m_totalPages` / `m_currentPageIndex` — 分页进度追踪
