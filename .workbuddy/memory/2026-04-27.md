## 架构重构（提交 17a84c9）
### P0 修复
- **Tab 子窗口加 parent**：`m_antennaDeviceWindow = new AntennaDeviceWindow(this)` 等 4 个 Tab 子窗口
- **创建 NetworkManager 类**：统一封装 TCP Server / Client / VNA 三种连接
- **MainWindow 不再直接操作 QTcpSocket**：移除 `m_tcpServer/m_clients/m_clientSocket/m_vnaSocket`
- 新增 `ConnectionHandle` 结构体封装连接信息（id/type/peerInfo/connected）

### 架构设计
```
MainWindow (主界面)
├── NetworkManager *m_network          ← 统一网络层
│   ├── TCP Server（服务器模式，接受外部客户端）
│   ├── TCP Client（客户端模式，主动连接外部服务器）
│   └── TCP VNA（矢网仪器连接）
│
├── Tab 子窗口（有 parent=this）
│   ├── AntennaDeviceWindow    → sendRawCommandToServer → NetworkManager::sendToServer
│   ├── TemperatureInfoWindow  → 同上
│   ├── PowerVoltageWindow     → 纯UI，无网络操作
│   └── ElectronicDeviceWindow → 纯UI，无网络操作
│
└── 菜单弹窗子窗口（有 parent=this）
    ├── AutoTestWindow         → connectToHostRequested → NetworkManager::connectToVna
    └── InstrumentControlWindow
```
